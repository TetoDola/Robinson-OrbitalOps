# Robinson — Clone de Terminal Industries

Clone statique de [terminal-industries.com](https://terminal-industries.com/) généré le 4 juillet 2026.

## Lancer le site

```bash
cd C:\Users\0303511D\Desktop\Robinson
npm run dev
```

Ouvrir http://localhost:3000

## Structure

- `src/components/` — 11 sections (Header, Hero, Marquee, Bridge, YOS, Benefits, Quote, HowItWorks, Contact, CTA, Footer)
- `public/fonts/` — SuisseIntl (Regular + Medium)
- `public/images/brands/` — logos clients
- `public/videos/` — vidéo hero bridge
- `docs/research/` — HTML source + topologie de page

## Limites connues

Sans browser MCP, ce clone reproduit la structure, le contenu, les couleurs et les assets principaux. Les animations scroll-driven avancées du site original (odomètre animé, transitions Lenis, etc.) sont simplifiées.
